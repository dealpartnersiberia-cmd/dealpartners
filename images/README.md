# images/

Carpeta para los recursos gráficos de DealPartners.

## Imágenes recomendadas

Añade aquí los siguientes archivos para completar el proyecto:

| Archivo                  | Uso                                              | Tamaño sugerido  |
|--------------------------|--------------------------------------------------|------------------|
| `logo.svg`               | Logotipo principal (versión vectorial)           | —                |
| `logo-dark.svg`          | Logotipo para fondos oscuros                     | —                |
| `favicon.ico`            | Favicon del navegador                            | 32×32 px         |
| `og-image.jpg`           | Imagen para redes sociales (Open Graph)          | 1200×630 px      |
| `hero-bg.jpg`            | Fondo opcional para la sección hero              | 1920×1080 px     |
| `team-1.jpg`             | Foto equipo / asesor 1                           | 600×800 px       |
| `team-2.jpg`             | Foto equipo / asesor 2                           | 600×800 px       |
| `pattern-gold.svg`       | Textura / patrón decorativo dorado               | —                |

## Cómo referenciar desde HTML

```html
<img src="images/logo.svg" alt="DealPartners" width="180">
```

## Cómo referenciar desde CSS

```css
background-image: url('../images/hero-bg.jpg');
```

## Recursos gratuitos recomendados

- Fotos: https://unsplash.com (busca "office", "handshake", "business meeting")
- Iconos SVG: https://heroicons.com · https://lucide.dev
- Optimización: https://squoosh.app (comprime antes de subir)
